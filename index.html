Introdução

Podemos escrever texto diretamente no código HTML. Não devemos! É porque assim ele fica "sem sentido" no documento.

Modo de usar

Usamos o programa seguindo os seguintes passos:

  1. Abrir
  2. Usar o Programa
  3. Fechar

Exemplo de código

Este é um exemplo de um programa "hello world" escrito em Java. Note que temos um único médoto estático chamado main() e que recebe parâmetros da linha de comando (que não são usados).

Para rodar o programa digite java Programa no prompt de comando. 

 
public static void main(String args[]) {
    System.out.println("Hello World!");
    System.out.println("Olá, Mundo!");
}
